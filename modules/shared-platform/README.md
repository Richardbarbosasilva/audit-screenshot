# Shared Platform

Camada transversal do repositório operacional do LeakGuard.

## Responsabilidade

- Semaphore
- Ansible
- inventários
- cronogramas operacionais
- integração com backup/retenção

Hoje essa camada está majoritariamente em:

- `infra/ansible/`
- `infra/semaphore/`
- runbooks do root
