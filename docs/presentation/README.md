# Diagramas de Apresentacao

Este diretório concentra a versão inicial dos diagramas de apresentação do projeto.

## Diagramas

- `01-fluxo-operacional.md`
  - foco no plano de controle
  - onde entram `GitHub`, `Semaphore`, `Ansible`, `WinRM`, `ShareX` e `ScreenshotAuditAgent`

- `02-fluxo-produto.md`
  - foco no produto e no plano de dados
  - onde entram `FastAPI`, `PostgreSQL`, `Redis`, `worker`, `MinIO` e a futura interface de revisão

- `03-visao-geral.md`
  - visão macro de ponta a ponta
  - bom para abertura de apresentação e alinhamento executivo

## Como usar

1. Validar o fluxo e os rótulos com o time.
2. Exportar para `SVG/PNG` com Mermaid ou draw.io.
3. Se quiser uma versão mais visual, usar estes arquivos como base para a arte final.

## Observação

Os diagramas foram desenhados para leitura rápida em apresentação.
Eles priorizam clareza e narrativa, não detalhamento exaustivo de infraestrutura.
